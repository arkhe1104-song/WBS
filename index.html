Agent Orchestrator · v1.0

WBS·IA 주간 분석 에이전트
메인 오케스트레이터

매주 갱신되는 WBS, IA_BO, IA_FO, IF Excel 파일을 자동 분석하여 크리티컬 리스크를 식별하고, 내부 관계자용 HTML 보고서·대시보드 4종을 생성하며 발주처 제출용 PDF를 자동 변환합니다.
특히 도메인별 상세 보고서에는 아이나(AINA) AI 챗봇과 텔레그램이 연동되었습니다.

단위는 반드시 명시합니다 (건, MD, %p 등).
